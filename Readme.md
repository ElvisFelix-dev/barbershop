## Iniciar

1. Clonar o repo
   ```sh
   git clone https://github.com/ElvisFelix-dev/barbershop
   ```
2. Abra a pasta
   ```sh
   cd barbershop

3. Instalando NPM packages
   ```sh
   npm install
   ```

4. Iniciar
   ```sh
   npm run dev
   ```
